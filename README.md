# Organiseret startpage

A startpage with customizable links, and full keyboard support for navigating settings and opening links.
Built 100% with vanilla JavaScript, HTML and CSS, no dependencies or packages.

## On running as local file

Sadly, due to CORS restrictions, running it locally will currently break the background pattern, but should otherwise work.

To do local development, run a local server (For example VSCode's [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer))
